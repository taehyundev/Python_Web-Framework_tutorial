# Django란 무어일까

Django는 웹을 띄울 수 있게 도와주는 하나의 프레임워크이다.


## Django를 사용하는 방법

pip 를 통해 django 를 설치해준다.

커맨드창에 아래와 같은 명령행을 적어준다.


```
pip install django
```

그리고 작업을 시행할 폴더로 이동한뒤에

```
django-admin startproject mysite
```
라고 적어준다.

그럼 mysite 라는 폴더가 생기게 된다.

```
python manage.py runserver
```
라고 적어서 서버를 실행한다.
